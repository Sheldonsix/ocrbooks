# ocrbooks
将一些稀有的扫描版电子书通过OCR识别转换为TXT/EPUB/MOBI格式

### 生成PDF

```
cd 目录
pandoc --from markdown+east_asian_line_breaks --css ../epub.css *.md -o 书本.epub
```
